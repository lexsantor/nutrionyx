# Deviations

Overrides of LPEF Standards for this project. One file per deviation, from the [deviation-record template](https://github.com/lexsantor/lpef/blob/v0.2.0/templates/documents/deviation-record.md).

Rules (normative source: LPEF GOVERNANCE.md R10):

- Every deviation names the exact rule it overrides, justifies it for this project specifically, and expires.
- The Constitution cannot be deviated from.
- Active deviations are reviewed at each LPEF release; recurring deviations are evidence the framework rule is wrong and feed amendments.

No active deviations.
