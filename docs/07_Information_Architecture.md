# 07. Information Architecture

## Principles

-   Mobile first.
-   Progressive disclosure.
-   Consistent navigation.
-   One primary action per screen.

## Nutritionist Workspace

Dashboard ├── Patients ├── Assessments ├── Treatments ├── Diet Plans ├──
Exercise ├── Medication ├── Reports └── Settings

## Patient Workspace

Home ├── Today ├── Progress ├── Measurements ├── Medication ├── Diet ├──
Exercise ├── Documents └── Profile

## Navigation Rules

-   Maximum three navigation levels.
-   Frequently used actions accessible within two clicks.
-   Persistent navigation.
-   Context-aware dashboards.

## UX Rules

-   Empty states everywhere.
-   Skeleton loading.
-   Inline validation.
-   Autosave.
-   Optimistic updates where appropriate.
-   Responsive by default.
