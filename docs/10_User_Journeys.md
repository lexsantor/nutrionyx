# 10. User Journeys

## Journey 1: New Patient

Invitation → Account Activation → Authentication → Mandatory Assessment
→ Treatment Creation → Dashboard Unlock

Success: Patient reaches first meaningful value within the first
session.

------------------------------------------------------------------------

## Journey 2: Daily Follow-up

Open dashboard → Review today's tasks → Log weight → Log measurements →
Confirm medication → Review progress

------------------------------------------------------------------------

## Journey 3: Nutritionist Consultation

Open patient → Review timeline → Compare progress → Update treatment →
Save review → Schedule next appointment

------------------------------------------------------------------------

## UX Success Metrics

-   Assessment completion \<15 min
-   Daily check-in \<2 min
-   Treatment update \<5 min
-   Zero unnecessary navigation
